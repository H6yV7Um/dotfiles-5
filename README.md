
# README

Save my dot files.

## Install

```
$ git clone git@github.com:hotoo/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ make install
```

## Uninstall

```
$ cd ~/.dotfiles
$ make uninstall
```
